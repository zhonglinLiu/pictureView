# 介绍

图片查看组件
# 使用
    Wheel.init({playZoom:2})
# 参数
## playZoom
+ Type: Number
+ Default: 1

显示图片的缩放比例

## playElement
+ Type: String
+ Default:#wheel-show

图片列表父元素

## barElement
+ Type: String
+ Default: #wheel-bar

底部图片列表父元素

## leftElement
+ Type: String
+ Default: #wheel-left

左翻页元素

## rightElement
+ Type: String
+ Default: #wheel-right

又翻页元素

## view
+ Type: String
+ Default: #view

当前图片元素

## moveCallback(index,images)
+ index
    + Type: Number
    + 图片索引
+ images
    + Type: Array
    + 图片数组

图片切换后执行

##update(data)

更新dom
